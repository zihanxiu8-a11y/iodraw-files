```mermaid
@startuml
title 事件驱动架构流程图（Outbox模式 + 幂等消费）

package "业务操作" {
  [借阅]
  [归还]
  [点赞帖子]
  [评论帖子]
  [关注用户]
}

package "事务内发布 (DomainEventPublisher)" {
  [构造 DomainEventEnvelope]
  [写入 Outbox 表 (status=PENDING)]
  [注册事务后回调]
}

rectangle "数据库事务边界" as Tx #LightGray

package "RabbitMQ" {
  [domain.event.exchange (Topic Exchange)]
  [notification.event.queue]
  [recommendation.event.queue]
  [死信交换机 DLX]
  [domain.event.dlq]
}

package "事件消费者" {
  [NotificationEventConsumer] as NC
  [RecommendationEventConsumer] as RC
}

package "幂等与重试" {
  [EventConsumptionService (幂等表检查)]
  [定时任务 retryPendingPublishes]
}

package "后置处理" {
  [发送 WebSocket 通知]
  [持久化通知到数据库]
  [刷新推荐缓存]
  [重新计算推荐]
}

' 连线
[借阅] -down-> [构造 DomainEventEnvelope]
[归还] -down-> [构造 DomainEventEnvelope]
[点赞帖子] -down-> [构造 DomainEventEnvelope]
[评论帖子] -down-> [构造 DomainEventEnvelope]
[关注用户] -down-> [构造 DomainEventEnvelope]

[构造 DomainEventEnvelope] -down-> [写入 Outbox 表 (status=PENDING)]
[写入 Outbox 表 (status=PENDING)] -down-> [注册事务后回调]

Tx ..> [写入 Outbox 表 (status=PENDING)]

[注册事务后回调] -down-> [domain.event.exchange (Topic Exchange)] : afterCommit 发送

[domain.event.exchange (Topic Exchange)] -down-> [notification.event.queue] : 绑定 routingKey\ndomain.borrowing.*\ndomain.social.*
[domain.event.exchange (Topic Exchange)] -down-> [recommendation.event.queue] : 绑定 routingKey\ndomain.borrowing.*

[notification.event.queue] -down-> NC
[recommendation.event.queue] -down-> RC

NC -down-> [EventConsumptionService (幂等表检查)]
RC -down-> [EventConsumptionService (幂等表检查)]

[EventConsumptionService (幂等表检查)] -down-> [发送 WebSocket 通知]
[EventConsumptionService (幂等表检查)] -down-> [持久化通知到数据库]
[EventConsumptionService (幂等表检查)] -down-> [刷新推荐缓存]
[EventConsumptionService (幂等表检查)] -down-> [重新计算推荐]

[定时任务 retryPendingPublishes] ..> [domain.event.exchange (Topic Exchange)] : 扫描 FAILED/PENDING\n重试投递

[domain.event.exchange (Topic Exchange)] -right-> [死信交换机 DLX] : 消息异常/超时
[死信交换机 DLX] -right-> [domain.event.dlq]

note top of [EventConsumptionService (幂等表检查)]
  幂等表包含唯一约束
  (event_id, consumer_name)
  保证每条消息只处理一次
end note

@enduml
```
